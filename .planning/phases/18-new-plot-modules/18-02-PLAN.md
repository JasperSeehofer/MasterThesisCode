**Function 1: `plot_h0_convergence()`** (per D-10, D-11, D-12):

```python
def plot_h0_convergence(
    h_values: npt.NDArray[np.float64],
    event_posteriors: list[npt.NDArray[np.float64]] | npt.NDArray[np.float64],
    *,
    true_h: float | None = None,
    subset_sizes: list[int] | None = None,
    seed: int = 42,
    level: float = 0.68,
    ax: None = None,  # ignored, creates own 2-panel figure
) -> tuple[Figure, npt.NDArray[np.object_]]:
```

Implementation:
- Default subset_sizes = [1, 5, 10, 25, 50, 100], capped at len(event_posteriors) per D-11
- Two-panel layout: `fig, (ax_post, ax_ci) = get_figure(nrows=1, ncols=2, preset="double")`
- `rng = np.random.default_rng(seed)` per D-12
- For each subset_size N:
  - Random sample N indices from range(len(event_posteriors)) without replacement
  - Combine posteriors: `combined = np.prod([event_posteriors[i] for i in indices], axis=0)` -- use log-sum-exp for stability: `log_combined = np.sum([np.log(np.maximum(p, 1e-300)) for ...], axis=0)`, then `combined = np.exp(log_combined - log_combined.max())`
  - Normalize: `combined /= np.trapezoid(combined, h_values)`
  - Plot on ax_post with color from CYCLE (cycling), label=f"N={N}"
  - Compute 68% CI width using `_credible_interval_width()` private helper
- Left panel (ax_post): posterior curves, xlabel=LABELS["h"], ylabel="Posterior density", legend
- Right panel (ax_ci): CI width vs N. Plot data points + 1/sqrt(N) reference curve scaled to match first point. xlabel="Number of events", ylabel=r"68\% CI width"
- If true_h provided: ax_post.axvline(true_h, color=TRUTH, linestyle="--", label="Truth")
- Return `fig, np.array([ax_post, ax_ci], dtype=object)`

Private helper `_credible_interval_width()`:
```python
def _credible_interval_width(
    h_values: npt.NDArray[np.float64],
    posterior: npt.NDArray[np.float64],
    level: float = 0.68,
) -> float:
```
- Normalize posterior: `posterior = posterior / np.trapezoid(posterior, h_values)`
- Compute CDF: `cdf = np.cumsum(posterior * np.gradient(h_values))`, normalize to 1
- Interpolate: `lo = np.interp((1-level)/2, cdf, h_values)`, `hi = np.interp((1+level)/2, cdf, h_values)`
- Return `float(hi - lo)`

**Function 2: `plot_detection_efficiency()`** (per D-13, D-14, D-15):

```python
def plot_detection_efficiency(
    variable: npt.NDArray[np.float64],
    detected: npt.NDArray[np.bool_],
    *,
    bins: int = 20,
    confidence: float = 0.68,
    xlabel: str | None = None,
    ax: Axes | None = None,
) -> tuple[Figure, Axes]:
```

Implementation:
- If ax is None: `fig, ax = get_figure(preset="single")`; else `fig = _fig_from_ax(ax)`
- Compute bin edges: `np.linspace(variable.min(), variable.max(), bins + 1)`
- Bin centers: `0.5 * (edges[:-1] + edges[1:])`
- Count injected per bin: `n_inj = np.histogram(variable, bins=edges)[0]`
- Count detected per bin: `n_det = np.histogram(variable[detected], bins=edges)[0]`
- Mask empty bins: `mask = n_inj > 0`
- Efficiency: `np.where(mask, n_det / n_inj, np.nan)` per RESEARCH pitfall 5
- Wilson CI: `from astropy.stats import binom_conf_interval` then `ci = binom_conf_interval(n_det, n_inj, confidence_level=confidence, interval='wilson')` per D-14
- Plot efficiency as step line: `ax.step(bin_centers, efficiency, where='mid', color=CYCLE[0], linewidth=1.5)`
- Shaded CI band: `ax.fill_between(bin_centers, ci[0], ci[1], alpha=0.3, color=CYCLE[0], step='mid')`
- xlabel: use provided or default to LABELS["z"]. ylabel: r"$P_\mathrm{det}$"
- `ax.set_ylim(-0.05, 1.05)`
- Return `fig, ax`

Use `np.trapezoid` (not deprecated `np.trapz`) per Phase 17 decision.
