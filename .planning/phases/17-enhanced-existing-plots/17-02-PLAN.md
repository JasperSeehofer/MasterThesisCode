Plot at bin centers as line: `ax2.plot(bin_centers, fraction, color=MEAN, linewidth=1.5, label="Detection fraction")`. Set `ax2.set_ylabel("Detection fraction")`, range [0, 1].

    - Combine legends: `lines1, labels1 = ax.get_legend_handles_labels()`, `lines2, labels2 = ax2.get_legend_handles_labels()`, `ax.legend(lines1 + lines2, labels1 + labels2)`.
    - Return `(fig, ax)` -- primary axes.
