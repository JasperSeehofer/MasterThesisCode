**Phase 2 -- Invoke /physics-change protocol:**

Present to user:
- Old formula: `power_spectral_density_a_channel()` at `LISA_configuration.py:66-84` returns only instrumental noise (8 * sin^2(...) * [S_OMS * (...) + 2 * (...) * S_TM])
- New formula: Same instrumental noise PLUS S_c(f) = A * f^(-7/3) * exp(-(f/f1)^alpha) * 0.5 * (1 + tanh(-(f - fk) / f2)), where f1 = 10^(a1*log10(T_obs_sec) + b1), fk = 10^(ak*log10(T_obs_sec) + bk)
- Reference: LDC parameterization of Cornish & Robson (2017) arXiv:1703.09858; constants from constants.py:74-81. NOTE: The comment in constants.py cites arXiv:2303.15929 Eq. 17, but that paper does not contain a confusion noise formula -- the actual source is Cornish & Robson / Robson et al. (2019) arXiv:1803.01944 Eq. 14. Flag this discrepancy for user decision.
- Dimensional analysis: S_c has units Hz^(-1) (strain PSD). A [Hz^(-1) * Hz^(7/3)] * f^(-7/3) [Hz^(-7/3)] = Hz^(-1). Exponential and tanh are dimensionless. f1 and fk have units Hz (from power-law fit of T_obs in seconds).
- Limiting case: As f -> infinity, exp(-(f/f1)^1.8) -> 0 and tanh(-inf) -> -1, so S_c -> 0 (confusion noise vanishes at high frequencies, physically correct)

WAIT for user approval before proceeding to Phase 3.

   ```python
   # LDC parameterization of Cornish & Robson (2017), arXiv:1703.09858, Eq. (3)
   # Continuous T_obs dependence from LISA Data Challenge toolbox
   def _confusion_noise(
       self, frequencies: npt.NDArray[np.float64]
   ) -> npt.NDArray[np.float64]:
       xp = _get_xp(frequencies)
       t_obs_sec = self.t_obs_years * YEAR_IN_SEC
       f1 = 10.0 ** (LISA_PSD_A1 * xp.log10(t_obs_sec) + LISA_PSD_B1)
       fk = 10.0 ** (LISA_PSD_AK * xp.log10(t_obs_sec) + LISA_PSD_BK)
       return (  # type: ignore[no-any-return]
           LISA_PSD_A
           * frequencies ** (-7.0 / 3.0)
           * xp.exp(-(frequencies / f1) ** LISA_PSD_ALPHA)
           * 0.5
           * (1.0 + xp.tanh(-(frequencies - fk) / LISA_PSD_F2))
       )
   ```

4. Modify `power_spectral_density_a_channel` to conditionally add confusion noise:
   ```python
   def power_spectral_density_a_channel(
       self, frequencies: npt.NDArray[np.float64]
   ) -> npt.NDArray[np.float64]:
       xp = _get_xp(frequencies)
       instrumental = (
           8
           * xp.sin(2 * xp.pi * frequencies * L / C) ** 2
           * (
               self.S_OMS(frequencies) * (xp.cos(2 * xp.pi * frequencies * L / C) + 2)
               + 2
               * (
                   3
                   + 2 * xp.cos(2 * xp.pi * frequencies * L / C)
                   + xp.cos(4 * xp.pi * frequencies * L / C)
               )
               * self.S_TM(frequencies)
           )
       )
       if self.include_confusion_noise:
           instrumental = instrumental + self._confusion_noise(frequencies)
       return instrumental  # type: ignore[no-any-return]
   ```
