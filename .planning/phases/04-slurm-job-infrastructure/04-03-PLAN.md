```bash
DATESTAMP=$(date +%Y%m%d)
RUN_DIR="$WORKSPACE/run_${DATESTAMP}_seed${SEED}"
```

**Job submission chain (per D-06, SLURM-04):**
Submit three jobs using `sbatch --parsable`, passing `--output` and `--error` on the command line (not in SBATCH headers, per Research Pitfall 1):

1. Simulate array job:
```bash
SIM_JOB=$(sbatch --parsable \
    --array="0-$((TASKS - 1))" \
    --output="$RUN_DIR/logs/simulate_%A_%a.out" \
    --error="$RUN_DIR/logs/simulate_%A_%a.err" \
    --export=ALL,RUN_DIR="$RUN_DIR",BASE_SEED="$SEED",SIM_STEPS="$STEPS" \
    "$CLUSTER_DIR/simulate.sbatch")
```

2. Merge job (depends on all simulate tasks):
```bash
MERGE_JOB=$(sbatch --parsable \
    --dependency="afterok:$SIM_JOB" \
    --output="$RUN_DIR/logs/merge_%j.out" \
    --error="$RUN_DIR/logs/merge_%j.err" \
    --export=ALL,RUN_DIR="$RUN_DIR" \
    "$CLUSTER_DIR/merge.sbatch")
```

3. Evaluate job (depends on merge):
```bash
EVAL_JOB=$(sbatch --parsable \
    --dependency="afterok:$MERGE_JOB" \
    --output="$RUN_DIR/logs/evaluate_%j.out" \
    --error="$RUN_DIR/logs/evaluate_%j.err" \
    --export=ALL,RUN_DIR="$RUN_DIR" \
    "$CLUSTER_DIR/evaluate.sbatch")
```

**Summary output:**
Print a summary table showing:
- Run directory path
- Simulate job ID + array range
- Merge job ID + dependency
- Evaluate job ID + dependency
- A note about checking `sacct -j $SIM_JOB` for status

Example output format:
```
Pipeline submitted:
  Run directory: /path/to/run_20260327_seed42
  Simulate:  JOB_ID (array 0-99)
  Merge:     JOB_ID (after simulate)
  Evaluate:  JOB_ID (after merge)
Monitor: sacct -j SIM_JOB,MERGE_JOB,EVAL_JOB
```

**Query failed tasks:**
Use `sacct` to find failed array task indices (per Research sacct query):
```bash
FAILED_TASKS=$(sacct --array --jobs="$JOB_ID" \
    --state=FAILED,TIMEOUT,NODE_FAIL,OUT_OF_MEMORY \
    --format=JobID%30 \
    --noheader --parsable2 \
    | grep -oP '^\d+_\K\d+' \
```

- `$RUN_DIR/simulations/cramer_rao_bounds_simulation_${IDX}.csv`
- `$RUN_DIR/simulations/undetected_events_simulation_${IDX}.csv`
- `$RUN_DIR/run_metadata_${IDX}.json`

Use `rm -f` (no error if file doesn't exist). Print each deleted file.

**Resubmit failed tasks:**
Convert the list of failed task IDs to a comma-separated string (e.g., `2,5,17`) and resubmit:
```bash
FAILED_ARRAY=$(echo "$FAILED_TASKS" | tr '\n' ',' | sed 's/,$//')

# Extract original BASE_SEED and SIM_STEPS from an existing run_metadata file
# (any surviving metadata file will have the original seed info)
CLUSTER_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"

echo "Resubmitting failed tasks: $FAILED_ARRAY"
RESUB_JOB=$(sbatch --parsable \
    --array="$FAILED_ARRAY" \
    --output="$RUN_DIR/logs/simulate_%A_%a.out" \
    --error="$RUN_DIR/logs/simulate_%A_%a.err" \
    --export=ALL,RUN_DIR="$RUN_DIR",BASE_SEED="$BASE_SEED",SIM_STEPS="$SIM_STEPS" \
    "$CLUSTER_DIR/simulate.sbatch")
```
