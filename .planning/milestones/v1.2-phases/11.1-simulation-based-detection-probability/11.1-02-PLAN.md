```python
class SimulationDetectionProbability:
    """Simulation-based detection probability from injection campaign data.

    Loads raw injection CSVs (z, M, phiS, qS, SNR, h_inj), applies an SNR
    threshold at evaluation time, bins into P_det grids, and provides
    interpolated lookups.

    Per D-02: starts with P_det(z, M | h) by marginalizing over sky angles.
    Per D-03: raw SNR stored; threshold applied here at evaluation time.
    Per D-06: interpolates P_det between h grid points.
    """
```

**Constructor** `__init__(self, injection_data_dir: str, snr_threshold: float, h_grid: list[float] | None = None)`:
1. Glob all CSV files matching `injection_h_*_task_*.csv` in `injection_data_dir` (also check for merged files `injection_h_*.csv`)
2. Group files by h value (extract from filename pattern `injection_h_{h_label}`)
3. Load and concatenate CSVs per h value using pandas
4. If `h_grid` is None, auto-detect from files; otherwise use provided grid
5. Store `self._snr_threshold = snr_threshold`
6. Store `self._h_grid = sorted(h_grid_values)`
7. For each h value: call `_build_grid(df, snr_threshold)` -> store in `self._interpolators: dict[float, RegularGridInterpolator]`

**`_build_grid(self, df: pd.DataFrame, snr_threshold: float)` -> `RegularGridInterpolator`:**
1. Per D-02: build P_det(z, M | h) by marginalizing over sky angles (simpler, less noisy)
2. Define z_bins: 30 linearly spaced from 0 to max(df["z"]) * 1.1
3. Define M_bins: 20 log-spaced from min(df["M"]) * 0.9 to max(df["M"]) * 1.1
4. Use `np.histogram2d` twice: once for all events, once for detected events (SNR >= threshold)
5. P_det grid = detected_counts / total_counts, with 0/0 -> 0.0 (conservative, per RESEARCH Pitfall 2)
6. Use bin centers (not edges) as grid coordinates
7. Return `RegularGridInterpolator((z_centers, M_centers), p_det_grid, method="linear", bounds_error=False, fill_value=0.0)`

**`detection_probability_with_bh_mass_interpolated(self, d_L, M_z, phi, theta, *, h: float)` -> float | NDArray:**
1. Convert d_L to z using `dist_to_redshift(d_L, h=h)` (import from physical_relations)
2. Convert M_z to M_true: `M_true = M_z / (1 + z)` (observer-frame to source-frame mass)
3. Call `_interpolate_at_h(z, M_true, h)` which handles h interpolation

**`detection_probability_without_bh_mass_interpolated(self, d_L, phi, theta, *, h: float)` -> float | NDArray:**
1. Convert d_L to z using `dist_to_redshift(d_L, h=h)`
2. For the "without BH mass" variant, marginalize over M: query the grid at the given z with M at several points and average, OR use a 1D z-only grid (simpler)
3. Recommended: build a separate 1D z-only grid `self._interpolators_1d` during construction, where P_det(z | h) = (# events with SNR >= threshold at z-bin) / (# total events at z-bin)
4. Call `_interpolate_at_h_1d(z, h)`

**`_interpolate_at_h(self, z, M, h)` -> float | NDArray:**
1. Find bracketing h values: h_low, h_high from self._h_grid
2. If exact match: return self._interpolators[h_exact](np.column_stack([z_arr, M_arr]))
3. Otherwise linear interpolation: t = (h - h_low) / (h_high - h_low), return (1-t) * P_low + t * P_high
4. For h outside grid range: clamp to nearest grid point

**Ensure pickle safety**: avoid lambda, use only basic types and scipy objects (RegularGridInterpolator is pickle-safe).

Type annotations: all methods fully typed per project conventions. Use `npt.NDArray[np.float64]` for array types.
