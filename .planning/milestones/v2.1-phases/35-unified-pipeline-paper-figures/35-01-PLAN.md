def compute_credible_interval(

Add the necessary imports at the top of `_helpers.py`: `import numpy as np` and `import numpy.typing as npt`.

Implementation uses cumulative per-step `np.trapezoid` (matching `paper_figures.py` style, per D-07):
- Normalize: `norm = np.trapezoid(posterior, h_values)`, return `(nan, nan)` if `norm <= 0`
- Build CDF: loop `for i in range(1, len(h_values)):` accumulating `np.trapezoid(p[i-1:i+1], h_values[i-1:i+1])`
- Normalize CDF: `cdf /= cdf[-1]`
- Interpolate: `lo = np.interp((1-level)/2, cdf, h_values)`, `hi = np.interp((1+level)/2, cdf, h_values)`
- Return `(float(lo), float(hi))`

NumPy-style docstring with Args/Returns sections. Reference: "per D-07 from phase 35 CONTEXT.md".
