<objective>
Add the `--catalog_only` CLI flag and per-event diagnostic CSV logging to the evaluation pipeline.

Purpose: Enable running the pipeline with f_i=1.0 (bypassing the completion term) to diagnose whether L_comp is the dominant bias source. The diagnostic CSV captures L_cat, L_comp, f_i, and combined likelihoods per (event, h) pair for post-hoc analysis.
Output: Modified arguments.py, main.py, bayesian_statistics.py with catalog_only support + diagnostic CSV writing; test file validating the new behavior.
</objective>
