<objective>
Make P_det grid bin counts configurable via CLI flags, increase defaults from 30/20 to 60/40, and add coverage validation that reports the fraction of events whose 4-sigma d_L bounds fall within the grid.

Purpose: Higher grid resolution reduces interpolation error in the detection probability lookup, and coverage validation ensures the grid spans the physically relevant parameter space for the observed events.

Output: Fully configurable P_det grid with coverage reporting, all tests passing.
</objective>
