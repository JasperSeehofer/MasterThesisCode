    ```python
    ) -> tuple[Figure, npt.NDArray[np.object_]]:
    ```

    - `injected` and `recovered` are dicts mapping parameter name -> array of values. `uncertainties` maps parameter name -> 1-sigma CRB error bars (optional).

    - Compute grid layout: `nrows = ceil(len(parameters) / ncols)`.
    - Create figure using `plt.figure(figsize=(7.0, 2.8 * nrows))` -- acceptable exception to `get_figure` per Pitfall 6 in RESEARCH.md. Use `GridSpec(nrows * 2, ncols, height_ratios=[3, 1] * nrows, hspace=0.05, wspace=0.35)`.
    - For each parameter:
      - **Main scatter panel** (`gs[row * 2, col]`):
        - Identity line: `ax_main.plot([lo, hi], [lo, hi], color=REFERENCE, linestyle="--", linewidth=1)` where `lo = min(injected[p].min(), recovered[p].min())`, `hi = max(...)`.
        - Scatter with optional error bars: if `uncertainties` provided, `ax_main.errorbar(injected[p], recovered[p], yerr=uncertainties[p], fmt=".", color=CYCLE[0], capsize=2, markersize=3, alpha=0.7)`. Else plain scatter.
        - y-axis label: `LABELS[label_key(p)]` + " (recovered)" on leftmost column only.
        - Hide x-tick labels (shared with residual below): `plt.setp(ax_main.get_xticklabels(), visible=False)`.
      - **Residual sub-panel** (`gs[row * 2 + 1, col]`, sharex with main, per D-12):
        - Compute `residual = recovered[p] - injected[p]`.
        - If uncertainties: `ax_resid.errorbar(injected[p], residual, yerr=uncertainties[p], fmt=".", color=CYCLE[0], capsize=2, markersize=3, alpha=0.7)`. Else plain scatter.
        - Zero line: `ax_resid.axhline(0, color=REFERENCE, linestyle="--", linewidth=1)`.
        - x-axis label: `LABELS[label_key(p)]` + " (injected)" on bottom row only.
        - y-axis label: r"$\Delta$" on leftmost column only.
    - Collect all axes into ndarray for return.
    - Return `(fig, axes_array)` where axes_array is the ndarray of all axes.
