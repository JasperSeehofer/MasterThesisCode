```python
def plot_fisher_corner(
    covariance: npt.NDArray[np.float64],   # 14x14

    *,

    n_samples: int = 5000,
    seed: int = 42,
    ax: None = None,  # ignored, corner creates own figure
) -> tuple[Figure, npt.NDArray[np.object_]]:
```

Implementation:
- Default params = ["M", "mu", "a", "luminosity_distance", "qS", "phiS"] per D-07
- Extract sub-indices from PARAMETER_NAMES
- Build sub_cov and sub_mean via `np.ix_(indices, indices)`
- Generate samples: `rng = np.random.default_rng(seed)`, `samples = rng.multivariate_normal(sub_mean, sub_cov, size=n_samples, check_valid='warn')`

- Extract axes: `axes = np.array(fig.axes, dtype=object).reshape(n, n)`
- Return `fig, axes`
