    Create a standalone analysis script that assesses P_det grid quality:

    1. Load injection data:
       - Load all 262 CSVs from simulations/injections/ grouped by h-value
       - Use same file-grouping logic as SimulationDetectionProbability (regex on filename)
       - CSV columns: z, M, phiS, qS, SNR, h_inj, luminosity_distance

    2. Build 30x20 grid per h-value:
       - Use IDENTICAL binning logic to SimulationDetectionProbability._build_grid_2d():
         * dl_edges = np.linspace(0, 1.1*max(d_L), 31)
         * M_edges = np.geomspace(0.9*min(M), 1.1*max(M), 21)
       - Compute per-bin: n_total (total events), n_det (SNR >= 15), p_hat = n_det/n_total
       - This MUST match the existing grid construction exactly so quality flags are meaningful

    3. Compute Wilson 95% CIs per bin:
       - Use astropy.stats.binom_conf_interval(n_det, n_total, confidence_level=0.9545, interval='wilson')
       - For empty bins (n_total=0): set CI = [0, 0], p_hat = 0, flag as empty
       - Compute CI_half_width = (CI_upper - CI_lower) / 2 per bin
       - Flag bins with n_total < 10 as unreliable

    4. Compute summary statistics per h:
       - Number of empty bins, unreliable bins (n<10), reliable bins (n>=10)
       - Median CI half-width for reliable bins
       - Max CI half-width across all non-empty bins
       - Number of bins in detection boundary region (0.05 < p_hat < 0.95)
       - If boundary region has < 5 bins, also report stats for "P_det > 0" bins

    5. Build 15x10 grid per h-value:
       - Use SAME d_L and M ranges as the 30x20 grid (same dl_max, M_min, M_max)
       - dl_edges_coarse = np.linspace(0, dl_max, 16)
       - M_edges_coarse = np.geomspace(M_min, M_max, 11)
       - Compute per-bin: n_total, n_det, p_hat, Wilson CIs, CI_half_width
       - Flag unreliable bins (n<10)

    6. Grid comparison:
       - Per-h comparison table: bins_total, bins_unreliable, median_ci_halfwidth, boundary_bins
       - Interpolation error: build RegularGridInterpolator from 15x10 P_det, evaluate at 30x20 bin centers, compute |P_det_interp - P_det_fine| for bins where P_det_fine > 0
       - Report: median absolute error, max absolute error, fraction of bins with error > 0.05
       - Boundary region comparison: if sufficient bins exist (>= 5), compare CI half-widths in boundary region between 30x20 and 15x10

    7. Consistency checks (run automatically):
       - Verify CI_lower <= p_hat <= CI_upper for all non-empty bins
       - Verify 0 <= CI_lower and CI_upper <= 1 for all bins
       - Verify n_det <= n_total for all bins
       - Verify sum of n_total across bins = total events per h (no events lost in binning, accounting for events outside bin range)

    CRITICAL: The grid is in (d_L, M) space, NOT (z, M). Use the 'luminosity_distance' column from the CSV, not 'z'. The 'M' column is source-frame mass.

    Structure the script with clear functions. Include a __main__ block that runs the full analysis for all 7 h-values and prints summary tables.
