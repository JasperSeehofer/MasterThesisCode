**Section 7: Extending to 4D — adding M_z as observable**

Start from the d_L-only baseline (Plan 01, Section 6). The "with BH mass" channel adds a 4th GW observable: the redshifted (detector-frame) mass M_z.

The GW detector measures M_z = M * (1+z), not the source-frame mass M. The galaxy catalog provides source-frame mass estimates (M_gal, sigma_M_gal). This means we must transform between the two frames when combining GW and galaxy information.

Define coordinates:
- Source-frame: M [M_sun]
- Detector-frame: M_z = M * (1+z) [M_sun]
- Fractional: M_z_frac = M_z / M_z_det = M * (1+z) / M_z_det [dimensionless]

The 4D GW likelihood is: p(d_GW | phi, theta, d_L_frac, M_z_frac), a 4D multivariate Gaussian from the Fisher matrix.

**Section 8: Marginalization over source-frame mass**

The single-host numerator now includes an integral over source-frame mass M:

  Numerator = integral dz integral dM  [p_det * p_GW_4D(phi, theta, d_L_frac, M_z_frac(M,z)) * p_gal(z) * p_gal(M)]

where M_z_frac(M,z) = M * (1+z) / M_z_det.

**Key step: change of variables from M to M_z_frac.**

The galaxy mass prior p_gal(M) = N(M; M_gal, sigma_M_gal) is defined in source-frame mass. The GW likelihood is parameterized in M_z_frac. To perform the integral analytically, we need both in the same variable.

Option A (transform integration variable to M_z_frac):
  dM = (M_z_det / (1+z)) * d(M_z_frac)
  p_gal(M) dM = p_gal(M_z_frac * M_z_det / (1+z)) * (M_z_det / (1+z)) * d(M_z_frac)

  The Gaussian transforms as:
  N(M; M_gal, sigma^2) * dM -> N(M_z_frac; M_gal*(1+z)/M_z_det, (sigma*(1+z)/M_z_det)^2) * (M_z_det/(1+z)) * d(M_z_frac)

  Wait — apply the standard Gaussian change of variables:
  If M = M_z_frac * M_z_det / (1+z), then
  p_gal(M) |dM/d(M_z_frac)| = N(M_z_frac * M_z_det/(1+z); M_gal, sigma^2) * M_z_det/(1+z)

  Using the Gaussian scaling property N(ax+b; mu, s^2) = (1/|a|) * N(x; (mu-b)/a, (s/a)^2):
  = N(M_z_frac; M_gal*(1+z)/M_z_det, (sigma_M*(1+z)/M_z_det)^2) * M_z_det/(1+z) * ...

  Actually, let's be more careful. We have:
  p_Y(y) = p_X(g^{-1}(y)) * |dg^{-1}/dy|
  where Y = M_z_frac, X = M, g(M) = M*(1+z)/M_z_det, g^{-1}(M_z_frac) = M_z_frac * M_z_det/(1+z)
  |dg^{-1}/dy| = M_z_det / (1+z)

  So: p_{M_z_frac}(y) = N(y * M_z_det/(1+z); M_gal, sigma^2) * M_z_det/(1+z)

  The N(y * M_z_det/(1+z); M_gal, sigma^2) can be rewritten using the Gaussian identity:
  N(ay; mu, s^2) = (1/|a|) * N(y; mu/a, (s/a)^2)
  with a = M_z_det/(1+z):

  p_{M_z_frac}(y) = [1/(M_z_det/(1+z))] * N(y; M_gal*(1+z)/M_z_det, (sigma_M*(1+z)/M_z_det)^2) * M_z_det/(1+z)

  The M_z_det/(1+z) factors CANCEL:
  p_{M_z_frac}(y) = N(y; M_gal*(1+z)/M_z_det, (sigma_M*(1+z)/M_z_det)^2)

  THIS IS THE KEY RESULT: After proper change of variables, the galaxy mass prior in M_z_frac coordinates is simply a Gaussian with transformed mean and variance. **No leftover Jacobian factor.** The |dM/d(M_z_frac)| cancels with the 1/|a| from the Gaussian scaling property.

  Therefore: mu_gal_frac = M_gal * (1+z) / M_z_det, sigma_gal_frac = sigma_M * (1+z) / M_z_det
  And the integrand in M_z_frac coordinates is:
  p_GW_4D(phi, theta, d_L_frac, M_z_frac) * N(M_z_frac; mu_gal_frac, sigma_gal_frac^2)

  with NO additional /(1+z) Jacobian.

Derive this step-by-step with explicit algebra. This is the central result that determines whether the /(1+z) at line 679 is spurious.

**Section 9: Conditional decomposition and analytic marginalization**

Apply Bishop (2006) PRML Eq. 2.81-2.82 to decompose the 4D Gaussian:

  p_GW_4D(phi, theta, d_L_frac, M_z_frac) = p_GW_3D(phi, theta, d_L_frac) * p(M_z_frac | phi, theta, d_L_frac)

where:
- p_GW_3D = marginal 3D Gaussian with covariance = cov_4d[:3,:3]
- p(M_z_frac | obs) = N(M_z_frac; mu_cond, sigma2_cond) with
  mu_cond = mu_4[3] + cov_cross @ cov_obs_inv @ (x_obs - mu_4[:3])
  sigma2_cond = cov_4d[3,3] - cov_cross @ cov_obs_inv @ cov_cross

Now the M_z_frac integral is:

  integral d(M_z_frac) N(M_z_frac; mu_cond, sigma2_cond) * N(M_z_frac; mu_gal_frac, sigma_gal_frac^2)
  = N(mu_cond; mu_gal_frac, sigma2_cond + sigma_gal_frac^2)

by the standard Gaussian product identity. Call this result `mz_integral`.

**Section 10: Complete "with BH mass" numerator integrand**

Assembling everything, the numerator integrand is:

  p_det * p_GW_3D(phi, theta, d_L_frac) * mz_integral * p_gal(z)

with NO /(1+z) factor. The (1+z) dependence enters ONLY through mu_gal_frac and sigma_gal_frac.

**Verdict on /(1+z) at line 679:** The /(1+z) is SPURIOUS. It was the Jacobian |dM/d(M_z)| = 1/(1+z) from the old code (before analytic marginalization), but after the change of variables to M_z_frac coordinates, this Jacobian is absorbed into the Gaussian transformation. The current code double-counts it.

**Section 11: "With BH mass" denominator**

Derive the denominator from the same Bayesian framework:

The denominator accounts for the selection effect — the probability that a source from this galaxy would be detected. It is:

  Denominator = integral dz integral dM  p_det(d_L, M_z, phi, theta) * p_gal(z) * p_gal(M)

where p_det now depends on M_z = M*(1+z) because the "with BH mass" detection probability depends on mass.

Key question: Does the denominator need the same Jacobian / /(1+z) treatment as the numerator?

The answer depends on whether we change variables to M_z_frac in the denominator. Since p_det is evaluated at (d_L, M_z, phi, theta) and M_z = M*(1+z), the natural integration variable is M (source-frame), not M_z_frac:

  Denominator = integral dz integral dM  p_det(d_L(z,H0), M*(1+z), phi, theta) * p_gal(z) * N(M; M_gal, sigma_M^2)

In the code (lines 689-722), the denominator uses MC sampling from p_gal(z) * p_gal(M) — which is correct if p_det is evaluated at M_z = M*(1+z). Check that the code does this correctly.

State explicitly what terms should and should not appear in the denominator, compared to the numerator:
- p_det: YES (selection correction)
- p_gal(z): YES (galaxy prior)
- p_gal(M): YES (galaxy mass prior)
- p_GW: NO (data likelihood integrates to 1)
- /(1+z): NO (denominator integrates over source-frame M directly, no coordinate transformation needed)
- mz_integral: NO (this was the result of marginalizing the GW likelihood over M_z_frac)

**Section 12: Limiting case — sigma_Mz -> infinity**

Take the limit where the GW measurement gives no information about M_z (infinite uncertainty).

In the 4D covariance, sigma_Mz -> infinity means cov_4d[3,3] -> infinity.

Show step by step:
1. sigma2_cond = cov_4d[3,3] - cov_cross @ cov_obs_inv @ cov_cross -> infinity (because cov_4d[3,3] dominates)
2. mz_integral = N(mu_cond; mu_gal_frac, sigma2_cond + sigma_gal_frac^2) -> 1/sqrt(2*pi*sigma2_cond) -> 0

But this goes to zero, not to a constant! The resolution is that the 4D GW likelihood p_GW_4D is normalized:
  integral d(M_z_frac) p_GW_4D = p_GW_3D * integral d(M_z_frac) p(M_z_frac|obs) = p_GW_3D

So when sigma2_cond -> infinity, the conditional becomes flat and the mass integral gives:
  integral d(M_z_frac) [flat] * N(M_z_frac; mu_gal_frac, sigma_gal_frac^2) = [flat] * 1 = 1/sqrt(2*pi*sigma2_cond)

Wait — this needs careful handling. The mz_integral approaches 1/sqrt(2*pi*sigma2_cond) for large sigma2_cond. But the 3D marginal p_GW_3D is INDEPENDENT of sigma2_cond (it is the marginal, not the conditional).

So the full numerator integrand goes as:
  p_det * p_GW_3D * (1/sqrt(2*pi*sigma2_cond)) * p_gal(z)

The denominator similarly: if p_det depends on M_z and sigma_Mz -> infinity, the M-integral of p_det * p_gal(M) gives the average p_det over the mass distribution, which is the same as the d_L-only p_det when mass information is uninformative.

The KEY point: the 1/sqrt(2*pi*sigma2_cond) factor appears in BOTH numerator and denominator (or in neither, depending on normalization). In the RATIO (numerator/denominator), it cancels, leaving the d_L-only expression.

Work through this limit carefully and show that it reproduces the Plan 01 d_L-only result.
