Old formula:
```
dl_centers = 0.5 * (dl_edges[:-1] + dl_edges[1:])

    (dl_centers,), p_det_1d, method="linear",
    bounds_error=False, fill_value=None,
```

New formula:
```
dl_centers = 0.5 * (dl_edges[:-1] + dl_edges[1:])
dl_centers_anchored = np.concatenate([[0.0], dl_centers])
p_det_1d_anchored = np.concatenate([[_P_MAX_EMPIRICAL_ANCHOR], p_det_1d])
return RegularGridInterpolator(
    (dl_centers_anchored,), p_det_1d_anchored, method="linear",
    bounds_error=False, fill_value=None,
)
# On [0, c_0): linear interp from (0, P_MAX) to (c_0, p̂(c_0)).
# At d_L = 0: p_det = _P_MAX_EMPIRICAL_ANCHOR (≈ 0.806).
# Above dl_centers[-1]: still linearly extrapolated, but explicitly clipped
# to 0 in detection_probability_without_bh_mass_interpolated_zero_fill (Phase 44).
```

Reference: `.gpd/phases/45-p-det-first-bin-asymptote-fix/45-01-SUMMARY.md` for the empirical derivation; Plan 45-01 JSON `scripts/bias_investigation/outputs/phase45/p_max_h_independence.json` for the numerical Wilson 95% CI; `pdet_asymptote.json` (Step 1b) for the original h_inj=0.73 evidence.

Dimensional analysis: `_P_MAX_EMPIRICAL_ANCHOR ∈ [0, 1]` dimensionless. dl_centers in Gpc throughout. `np.concatenate` preserves monotonicity since `0.0 < dl_centers[0] = c_0 > 0` always. Interpolator output `∈ [0, 1]` after `np.clip(..., 0, 1)` in the calling function (line 750).
