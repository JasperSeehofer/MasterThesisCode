<objective>
Empirically establish that the p_det asymptote near d_L → 0 is h_inj-independent, then derive the single h-independent scalar `p_max_empirical` (with Wilson 95% lower bound) that Plan 45-02 will prepend to `_build_grid_1d`'s grid.

Purpose: Phase 44 regression test `test_zero_fill_no_h_dependent_step_for_close_dL` requires the production interpolator to vary by < 0.20 across h ∈ {0.65, …, 0.85}. The Plan 45-02 fix (anchoring at d_L=0) must use a scalar derived once from the unrescaled injection campaign, not per-h. This plan computes that scalar with a traceable empirical justification, and verifies the h-independence assumption holds across the full injection campaign (T10 only checked h_inj=0.73).

Output: A standalone diagnostic script `scripts/bias_investigation/test_12_p_max_h_independence.py` and its JSON artifact `scripts/bias_investigation/outputs/phase45/p_max_h_independence.json` containing per-group and pooled empirical asymptotes plus a recommended scalar that Plan 45-02 will quote in its `[PHYSICS]` commit message and code comment.
</objective>
