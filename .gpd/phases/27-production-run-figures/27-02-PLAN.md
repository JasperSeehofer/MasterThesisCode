  <name>Task 1: Extract MAP, credible intervals, precision, and bias from combined posteriors</name>
  <files>.gpd/phases/27-production-run-figures/extracted_results.json</files>
  <action>
    Load the combined posteriors from `cluster_results/eval_corrected_full/combined_posterior.json` and `cluster_results/eval_corrected_full/combined_posterior_with_bh_mass.json`.

    For EACH variant (without BH mass, with BH mass):

    1. Read h_values and posterior arrays from the JSON
    2. Normalize the posterior: posterior_norm = posterior / np.trapezoid(posterior, h_values)
    3. Compute MAP: h_MAP = h_values[np.argmax(posterior)]
    4. Compute CDF by cumulative trapezoidal integration: CDF(h_i) = integral from h_min to h_i of p(h) dh
    5. Normalize CDF so CDF[-1] = 1.0
    6. CRITICAL: The posterior is extremely peaked (probability concentrated in 1-2 bins). To extract meaningful CIs, interpolate the CDF onto a fine grid:
       - h_fine = np.linspace(h_values[0], h_values[-1], 1000)
       - CDF_fine = np.interp(h_fine, h_values, CDF)
    7. Extract 68% CI: find h_16 and h_84 where CDF_fine = 0.16 and CDF_fine = 0.84
    8. Extract 95% CI: find h_025 and h_975 where CDF_fine = 0.025 and CDF_fine = 0.975
    9. Compute CI width: sigma_h = (h_84 - h_16) / 2
    10. Compute precision: precision_pct = sigma_h / h_MAP * 100
    11. Compute bias: bias_pct = (h_MAP - 0.73) / 0.73 * 100
    12. Also compute the median: h_median = h_fine[np.searchsorted(CDF_fine, 0.5)]

    Also extract from the combined posterior JSONs: n_events_total, n_events_used, n_events_excluded, strategy.

    Save all results to `.gpd/phases/27-production-run-figures/extracted_results.json`:
    ```json
    {
      "without_bh_mass": {
        "map_h": ...,
        "median_h": ...,
        "ci_68": [h_16, h_84],
        "ci_95": [h_025, h_975],
        "sigma_h": ...,
        "precision_pct": ...,
        "bias_pct": ...,
        "n_events_total": 534,
        "n_events_used": 531
      },
      "with_bh_mass": {
        "map_h": ...,
        "median_h": ...,
        "ci_68": [h_16, h_84],
        "ci_95": [h_025, h_975],
        "sigma_h": ...,
        "precision_pct": ...,
        "bias_pct": ...,
        "n_events_total": 534,
        "n_events_used": 527
      },
      "h_true": 0.73,
      "thesis_baseline": {
        "without_mass": {"map": 0.712, "sigma": 0.028, "precision_pct": 3.9},
        "with_mass": {"map": 0.742, "sigma": 0.013, "precision_pct": 1.8}
      }
    }
    ```

    Print a formatted comparison table to stdout showing production vs thesis baseline values.

    IMPORTANT: If the CI width equals exactly one grid spacing (e.g., 0.02 for the 23-point grid), the interpolation is not resolving the posterior. In that case, note this limitation and report the grid-limited CI.

    IMPORTANT: The with-BH-mass grid has only 8 points with irregular spacing (h = 0.6, 0.62, 0.626, 0.68, 0.78, 0.782, 0.834). The CI extraction may be unreliable. Report this caveat.
  </action>

    4. CI_68 is contained within CI_95
