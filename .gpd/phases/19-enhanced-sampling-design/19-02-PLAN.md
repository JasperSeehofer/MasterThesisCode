    **Step 1: Neyman-optimal allocation function**

    `neyman_allocation(p_det_grid, n_total_grid, n_targeted_total, min_per_bin=5) -> NDArray[int]`:
    - Input: Phase 18 P_det grid (15x10) and n_total grid per h-value; total targeted budget N_targeted
    - Compute per-bin Bernoulli std dev: sigma_k = sqrt(P_hat_k * (1 - P_hat_k))
      - For bins with P_det = 0 or P_det = 1: sigma_k = 0 (no variance to reduce)
      - For boundary bins (0.05 < P_det < 0.95): sigma_k ~ 0.15 to 0.50
    - Neyman-optimal: N_k proportional to sigma_k (Cochran 1977 Ch. 5; Owen 2013 Ch. 8)
      - N_k = floor(N_targeted * sigma_k / sum(sigma_k)) for sigma_k > 0
      - Impose minimum: N_k = max(N_k, min_per_bin) for bins with n_total > 0 in pilot
      - Distribute remainder (N_targeted - sum(N_k)) to highest-sigma bins
    - Output: integer allocation array same shape as p_det_grid, summing to N_targeted exactly

    **Step 2: Variance reduction factor computation**

    `variance_reduction_factor(p_det_grid, n_total_grid, allocation_grid, n_pilot) -> dict`:
    - Uniform baseline variance per boundary bin: Var_uniform = P*(1-P) / n_uniform_boundary
      where n_uniform_boundary = n_total * (N_new / N_total_uniform) for a hypothetical uniform campaign of size N_new = n_pilot + N_targeted
    - Stratified variance per boundary bin: Var_stratified = P*(1-P) / (n_pilot_boundary + N_k)
      where n_pilot_boundary is the pilot contribution to that bin
    - VRF per bin = Var_uniform / Var_stratified = (n_pilot_boundary + N_k) / n_uniform_boundary
    - Report: per-bin VRF for boundary bins, weighted average VRF, minimum VRF

    CRITICAL: The VRF computation must use ACTUAL Phase 18 per-bin counts, not generic formulas.
    Load injection CSVs and compute per-bin statistics directly. Use `analysis/grid_quality.py:build_grid_with_ci` or replicate its binning logic.

    **Step 3: Run for all 7 h-values on 15x10 grid**

    4. Compute VRF for boundary bins
    5. Store results: h, N_pilot, N_targeted, boundary_bins, VRF_per_bin, VRF_mean

    The pilot is the existing Phase 18 data (100% of current events). The targeted is an ADDITIONAL campaign of the same size, with 70% concentrated on boundary bins. Total future budget = current + targeted.

    **Step 1: Defensive mixture proposal specification**

    Add to `analysis/sampling_design.py`:

    `defensive_mixture_weight(z, M, alpha, boundary_bins, dl_edges, M_edges, h, n_targeted_grid) -> float`:
    - q(z, M) = alpha * p_uniform(z, M) + (1 - alpha) * g_targeted(z, M)
    - p_uniform(z, M): the current injection sampling density = emri_distribution(M, z) (unnormalized)
    - g_targeted(z, M): enhanced density in boundary bins
      - Map (z, M) to (d_L, M) using d_L = dist(z, h)
      - Find which bin (i, j) the point falls in
      - g_targeted proportional to N_k_targeted[i,j] / bin_area[i,j] (piecewise constant in bins)
    - Weight for injection i: w_i = p_uniform(z_i, M_i) / q(z_i, M_i)
    - For pilot samples (drawn from p_uniform): q = p_uniform, so w_i = 1
    - For targeted samples: w_i = p_uniform / q = 1 / [alpha + (1-alpha) * g_targeted / p_uniform]
    - Weight bound: max(w_i) <= 1/alpha = 1/0.3 ~ 3.33 (bounded by construction)
